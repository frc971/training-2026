# 2026 Training Tasks

This folder holds the task side of the 2026 software training track.

The lessons in the repo explain the concepts. These task pages tell students what to do with those concepts. They are meant to feel closer to real team work than a generic classroom exercise.

## Task flow

1. Complete the Java evaluation before software training.
2. Read the matching lesson page before attempting the task.
3. Complete the task in simulation or in a scoped branch/worktree as directed.
4. Ask for review only after you can explain what you changed and how you tested it.
